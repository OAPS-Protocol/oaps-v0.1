# Getting Started with OAPS v0.1

## For Auditors:
1. Install Python 3.8+
2. Run: `pip install web3`
3. Generate proof: `python tools/oaps-proof-generator.py your-audit.json`
4. Register: Open `tools/oaps-signer-dapp.html` in browser

## For Projects:
1. Request audit from OAPS-accredited auditor
2. Receive OAPS-compliant proof hash
3. Verify on-chain: https://sepolia.etherscan.io/address/YOUR_CONTRACT
