## üéØ Core Principle: Validity ‚â† Quality

OAPS proves **existence, integrity, and source** ‚Äî not correctness or safety.

| Validity (What OAPS Verifies)          | Quality (What OAPS Does NOT Verify)   |
|----------------------------------------|----------------------------------------|
| ‚úÖ Proof existence at specific time     | ‚ùå Audit correctness                    |
| ‚úÖ Data integrity (unaltered)           | ‚ùå Auditor skill/competence             |
| ‚úÖ Cryptographic signature              | ‚ùå System security                      |
| ‚úÖ On-chain timestamp                   | ‚ùå Finding accuracy                     |
